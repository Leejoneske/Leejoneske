### 01 · Business Intelligence Dashboard
**Stack:** Python · Django · Chart.js · PostgreSQL · AWS

A data analytics platform built for a regional SME, consolidating sales, inventory, and customer data into a single reporting interface. Includes role-based access control, scheduled report generation, and exportable data views.

- Reduced manual reporting time by approximately 70% post-deployment
- Integrated with third-party accounting APIs for real-time financial data sync
- Deployed on AWS EC2 with RDS backend and CloudFront distribution

### 02 · Multi-Vendor E-Commerce Platform
**Stack:** Node.js · MongoDB · Flutter (Mobile) · Firebase · Cloudflare

A scalable marketplace application supporting multiple merchant storefronts, order management, and a companion mobile app built in Flutter. Designed for the East African market with support for M-Pesa and card payments.

- Architected for horizontal scalability with stateless API design
- Mobile app available on Android with offline capability for low-connectivity environments
- Full admin panel with inventory management, analytics, and vendor onboarding flows

---

### 03 · Community Health Data System
**Stack:** PHP · SQLite · Apache · Supabase

A lightweight data collection and reporting tool deployed for a community health organisation. Designed to operate in low-resource environments with intermittent connectivity, with offline data sync when a connection is available.

- Prioritised simplicity and reliability over feature richness given field deployment constraints
- Trained non-technical staff on system use; produced end-user documentation
- Open-source components contributed back to the community health tech ecosystem

---

### 04 · ML-Assisted Content Moderation Tool *(Internal Research Project)*
**Stack:** Python · TensorFlow · PyTorch · Django REST Framework

An internal research prototype exploring automated content classification using fine-tuned transformer models. Built as a learning project to deepen applied ML skills and evaluate deployment feasibility for a client use case.

- Achieved ~89% classification accuracy on a custom-labelled dataset
- REST API wrapper allows integration with existing content management workflows
- Documented methodology and findings; available on request
